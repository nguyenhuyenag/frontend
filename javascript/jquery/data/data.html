Data

Phương thức .data() trong jQuery dùng để lưu và truy xuất dữ liệu tùy chỉnh (data) cho các phần tử HTML. Nó cho phép gán và truy xuất các giá trị dữ liệu không phải là thuộc tính (non-attribute data) cho các phần tử mà không ảnh hưởng đến các thuộc tính HTML.

Lưu ý: .data() chỉ lưu dữ liệu tạm thời trong bộ nhớ của jQuery và không ghi xuống các thuộc tính HTML. Có nghĩa dữ liệu sẽ xuất hiện trong nguồn HTML khi xem mã nguồn trang web.

Phạm vi lưu trữ: Dữ liệu được lưu trữ bằng .data() chỉ tồn tại trong bộ nhớ của jQuery khi trang web đang chạy. Nếu refresh hoặc đóng tab/trình duyệt, dữ liệu này sẽ bị mất.

Ví dụ

            // Set value
            $('#myElement').data('name', 'John Doe');
            $('#myElement').data('age', 30);

            // Get value
            let name = $('#myElement').data('name');
            let age = $('#myElement').data('age');
            console.log(name); // Output: "John Doe"
            console.log(age);  // Output: 30

            // Check data exits
            let hasData = $('#myElement').hasData();
            console.log(hasData); // true nếu có dữ liệu, false nếu không có dữ liệu

            // Remove Data
            $(element).removeData([key]);
            $('#myElement').removeData('name');
            // Nếu không chỉ định key, tất cả dữ liệu đã được lưu trữ trong phần tử sẽ bị xóa
            $('#myElement').removeData();
