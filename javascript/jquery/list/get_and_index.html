get()

Lấy phần tử theo chỉ số

Ví dụ

  • foo
  • bar
  • moon

index()

Lấy chỉ số của phần tử được chọn

Ví dụ

            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
                <li class="target">Target Item</li>
                <li>Item 3</li>
            </ul>

            let index = $('li.target').index();
            $('#output').text("Index of target item: " + index); // = 2 (đếm từ 0)
