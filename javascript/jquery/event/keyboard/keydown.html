keydown()

Bắt sự kiện liên quan đến việc nhấn phím (xem event.which.html)

            $(document).keydown(function(event) {
                if (event.which === 13) {
                    console.log("Enter key pressed.");
                }
            });

            $(document).on("keydown", function(event) {
                console.log("Key pressed: " + event.key);
            });
