param()

Là hàm được sử dụng để biến đổi một đối tượng JavaScript thành một chuỗi dữ liệu URL được mã hóa

            let params = { name: "John", age: 30, city: "New York" };
            let queryString = decodeURIComponent(jQuery.param(params));
            console.log(queryString); // name=John&age=30&city=New+York
