$.ready

Hàm $.ready là một hàm trong jQuery cho phép bạn chạy mã khi trang HTML đã tải xong. Hàm này nhận một hàm làm đối số, và hàm đó sẽ được chạy khi trang đã tải xong. 

            // Ví dụ 1
            $.when( $.ready ).then(function() {
                // Document is ready.
            });
            
            // Ví dụ 2
            $.when(
                $.getJSON("ajax/test.json"),
                $.ready
            ).done(function(data) {
                // Document is ready.
                // Value of test.json is passed as `data`
            });
