Load

Bắt sự kiện khi một phần tử đã được tải hoàn tất. Ví dụ: "img", "iframe",...

            $(selector).on("load", function() {
                // Xử lý khi phần tử được tải hoàn tất
            });

            // Ví dụ
            $("#myImage").on("load", function() {
                // Xử lý khi hình ảnh đã tải hoàn tất
                console.log("Hình ảnh đã tải hoàn tất!");
            });

Window on Load

            // Bắt sự kiện khi toàn bộ trang web đã tải hoàn tất: Hình ảnh, tệp đa phương tiện và các tài nguyên khác
            $(window).on("load", function() {

            });

Unload

Sự kiện unload được kích hoạt khi:

  • Window được đóng.
  • Refresh lại trang.
  • Click vào link tới trang khác.
  • Chạy một URL từ thanh address.
  • Click vào button next hay previous
            $(window).on("unload", function () {
                console.log("Reload trang");
            });
