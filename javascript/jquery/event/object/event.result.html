event.result

Truyền dữ liệu giữa các sự kiện được kích hoạt bởi cùng một sự kiện

Ví dụ

            $("button").on("click", function(event) {
                return "Hello";
            });
            $("button").on("click", function(event) {
                $("p").html(event.result); // Hello
            });
