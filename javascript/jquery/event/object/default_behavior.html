Default behavior

Hành vi mặc định (default behavior) của sự kiện DOM là hành vi tự nhiên mà một phần tử sẽ thực hiện. Ví dụ hành vi mặc định của một liên kết (thẻ a) khi click vào là chuyển hướng trình duyệt đến URL được chỉ định trong thuộc tính href.

event.preventDefault()

Khi gắn một trình xử lý sự kiện vào phần tử, ta có thể sử dụng event.preventDefault() để ngăn hành vi mặc định và thay thế bằng một hành vi tùy chỉnh khác

event.isDefaultPrevented()

Phương thức dùng để kiểm tra xem liệu trình xử lý sự kiện đã ngăn mặc định hành vi của sự kiện hay chưa.

Ví dụ

            $("a").on("click", function(event) {
                event.preventDefault(); // Ngăn chặn hành vi mặc định của liên kết
                if (event.isDefaultPrevented()) {
                    console.log("Default behavior prevented.");
                } else {
                    console.log("Default behavior not prevented.");
                }
            });
