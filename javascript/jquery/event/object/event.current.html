event.current

  • event.target: Trả về phần tử DOM mà sự kiện đã xảy ra. Thường là phần tử mà ta thực hiện hành động như nhấp chuột, nhập liệu,... Nó có thể khác với phần tử mà ta gắn bộ xử lý sự kiện.
  • event.currentTarget: Trả về phần tử DOM mà bộ xử lý sự kiện đang được gắn. Nó thường là phần tử mà ta đã gắn bộ xử lý sự kiện. Dù sự kiện đã xảy ra trên phần tử con, nhưng event.currentTarget sẽ là phần tử mà bạn đã đính kèm bộ xử lý sự kiện.

Ví dụ

            $("#parent").click(function (event) {
                console.log(event.target.id);           // In ra "child"
                console.log(event.currentTarget.id);    // In ra "parent" luôn, vì gắn xử lý lên phần tử cha
            });

            $("#child").click(function (event) {
                console.log(event.target.id);           // In ra "child" luôn, vì gắn xử lý lên phần tử con
                console.log(event.currentTarget.id);    // In ra "child" luôn, vì gắn xử lý lên phần tử con
            });
