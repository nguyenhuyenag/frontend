Data

event.data là một thuộc tính của đối tượng event trong JavaScript. Nó cho phép ta truyền dữ liệu vào trình xử lý sự kiện. 

            $(selector).on(eventType, eventData, handler);

Ví dụ

            $(".myButton").on("click", { name: "John", age: 30 }, function (event) {
                var data = event.data;
                console.log("Name: " + data.name);
                console.log("Age: " + data.age);
            });
