attr()

            Get thuộc tính: $(selector).attr(attributeName);
            Set thuộc tính: $(selector).attr(attributeName, value);
            let hrefValue = $('a').attr('href');

removeAttr()

        
Xóa thuộc tính khỏi các phần tử đã chọn

        
            $(selector).removeAttr(attributeName);
        

        
        

        

        
prop()

        
            Get => $(selector).prop(property);
            Get => $(selector).prop(property, function(index,currentvalue));
            Set => $(selector).prop({property:value, property:value,...});
        

        

        
        
val()

        
Get hoặc set giá trị của các phần tử đầu vào: input, select, textarea

        
        
        
        

        


        
Phân biệt giữa attr và prop

        
    
            
  • Cả .attr() và prop() đều được sử dụng để lấy hoặc gán giá trị của thuộc tính được chỉ định, nhưng .attr() trả về giá trị mặc định (original state) còn .prop() trả về giá trị hiện tại (current state).
    • 
            
  • .attr() thay đổi thuộc tính cho thẻ HTML đó (HTML Attributes)
    • 
            
  • .prop() thay đổi các thuộc tính cho thẻ HTML đó theo cây DOM. (DOM properties)
