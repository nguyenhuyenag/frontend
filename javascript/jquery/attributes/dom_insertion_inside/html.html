html()

Là một phương thức trong jQuery, được sử dụng để lấy hoặc đặt nội dung HTML của phần tử đã chọn. Khi sử dụng html() mà không truyền tham số, nó sẽ trả về nội dung HTML của phần tử đã chọn. Khi truyền một chuỗi nội dung HTML vào .html(), nó sẽ đặt nội dung HTML mới cho phần tử đã chọn.

Ví dụ

            <div class="demo-container">
                <div class="demo-box">Demonstration Box</div>
            </div>

            // Lấy nội dung HTML của phần tử div
            $( "div.demo-container" ).html();
            // Output
            <div class="demo-box">Demonstration Box</div>

            // Đặt nội dung HTML mới cho phần tử div
            $("div").html("<p>New HTML content</p>");
