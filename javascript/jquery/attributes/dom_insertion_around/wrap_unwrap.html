wrap()

Hàm wrap() dùng để bọc một phần tử cha (wrapper element) xung quanh một hoặc nhiều phần tử đã chọn.

Ví dụ

            // Với
            <p>This is paragraph 1</p>
            <p>This is paragraph 2</p>

            // Sử dụng
            $("p").wrap("<div class='wrapper'></div>");

            // Ta sẽ được
            <div class="wrapper">
                <p>This is paragraph 1.</p>
            </div>
            <div class="wrapper">
                <p>This is paragraph 2.</p>
            </div>

wrapAll()

Hàm sẽ bọc tất cả các phần tử được chọn bằng 1 phần tử cha

Ví dụ

            // Với
            <p>This is paragraph 1</p>
            <p>This is paragraph 2</p>

            // Sử dụng
            $("p").wrapAll("<div class='wrapper'></div>");

            // Ta sẽ được
            <div class="wrapper">
                <p>This is paragraph 1.</p>
                <p>This is paragraph 2.</p>
            </div>

wrapInner()

Ví dụ

            // Input
            <div id="myDiv">
                <p>This is a paragraph.</p>
                <button>Click me</button>
            </div>

            // Code
            $('#myDiv').wrapInner("<div class='wrapper'></div>");

            // Ouput
            <div id="myDiv">
                <div class="wrapper">
                    <p>This is a paragraph.</p>
                    <button>Click me</button>
                </div>
            </div>
