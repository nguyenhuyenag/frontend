replaceAll()

Phương thức .replaceAll() được dùng để thay thế một tập hợp các phần tử đã chọn bằng một phần tử khác hoặc nội dung.

Ví dụ

            <div class="container">
                <div class="inner first">Hello</div>
                <div class="inner second">And</div>
                <div class="inner third">Goodbye</div>
            </div>

            // Apply
            $("<h2>New heading</h2>").replaceAll(".inner");

            // Output
            <div class="container">
                <h2>New heading</h2>
                <h2>New heading</h2>
                <h2>New heading</h2>
            </div>

            // Or
            $(".first").replaceAll(".third");

            // Output
            <div class="container">
                <div class="inner second">And</div>
                <div class="inner first">Hello</div>
            </div>
