after()

Phương thức .after() được sử dụng để chèn phần tử, nội dung hoặc HTML sau một phần tử đã chọn. Điều này thay đổi cấu trúc của tài liệu HTML bằng cách thêm một phần tử mới vào sau phần tử đã chọn

Ví dụ

            // Cho
            <div class="container">
                <h2>Greetings</h2>
                <div class="inner">Hello</div>
                <div class="inner">Goodbye</div>
            </div>
            
            // Với
            $(".inner").after("<p>Test</p>");
            
            // Ta được
            <div class="container">
                <h2>Greetings</h2>
                <div class="inner">Hello</div>
                <p>Test</p>
                <div class="inner">Goodbye</div>
                <p>Test</p>
            </div>

Ta cũng có thể sử dụng .after() để chèn một phần tử DOM đã có vào sau một phần tử khác:

            let existingDiv = document.createElement("div");
            existingDiv.textContent = "Existing div content";
            $("p").after(existingDiv);

Passing a Function

Ta có thể custom giá trị chèn vào bằng cách

            $("p").after(function() {
                return "<div>" + this.className + "</div>";
            });
