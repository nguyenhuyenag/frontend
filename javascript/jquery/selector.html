Selector Example Description
* $("*") Tất cả các phần tử
#id $("#lastname") Phần tử có id="lastname"
.class $(".intro") Tất cả các phần tử có class="intro"
.class, .class $(".intro, .demo") Tất cả các phần tử có class="intro" hoặc "demo"
element $("p") Tất cả các thẻ <p>
el1,el2,el3 $("h1,div,p") Tất cả các thẻ <h1>, <div> và <p>
     
:first $("p:first") Thẻ <p> đầu tiên
:last $("p:last") Thẻ <p> cuối cùng
:even $("tr:even") Tất cả các <tr> có chỉ số chẵn (đếm từ 0)
:odd $("tr:odd") Tất cả <tr> có chỉ số chẵn
     
parent > child $("div > p") Tất cả các thẻ <p> con (cấp 1) của <div>
parent descendant $("div p") Tất cả các thẻ <p> nằm trong <div>
element + next $("div + p") Chọn thẻ <p> gần nhất nằm phía sau và ngang hàng (cùng cấp) với <div>
element ~ siblings $("div ~ p") Chọn tất cả các thẻ <p> ngang hàng phía sau <div>
element1.element2 $("p.intro") Chọn tất cả các thẻ <p> có class='intro'
     
:eq(index) $("ul li:eq(3)") Chọn phần tử thứ 4 (tính từ 0)
:gt(index) $("ul li:gt(3)") Những <li> có chỉ số ≥ 3 trong <ul>
:lt(no) $("ul li:lt(3)") Những <li> có chỉ số ≤ 3 trong <ul>
:not(selector) $("li:not(.highlight)") Chọn tất cả các <li> không chứa class="highlight"
     
[attribute] $("[href]") Tất cả phần tử có thuộc tính href
[attribute=value] $("[href='default.htm']") Tất cả phần tử mà href='default.htm'
[attribute!=value] $("[href!='default.htm']") Tất cả phần tử mà giá trị của href khác default.htm
[attribute$=value] $("[href$='.jpg']") Tất cả phần tử mà giá trị của href kết thúc bằng .jpg
[attribute|=value] $("[title|='tomorrow']") Tất cả phần tử có title='tomorrow', hoặc có dạng title='tomorrow-XXXX'
[attribute^=value] $("[title^='Tom']") Tất cả phần từ có title bắt đầu bằng "Tom"
[attribute~=value] $("[title~='hello']") Tất cả phần tử mà title chứa 'hello' (độc lập bởi dấu cách, 'helloo' không đúng)
[attribute*=value] $("[title*='hello']") Tương tự trên nhưng cả "hello" và "helloo" đều hợp lệ
[name=”value”][name2=”value2″] $("input[id][name$='man']") Chọn phần từ dựa trên nhiều thuộc tính
     
:input $(":input") All input elements
:text $(":text") All input elements with type="text"
:password $(":password") All input elements with type="password"
:radio $(":radio") All input elements with type="radio"
:checkbox $(":checkbox") All input elements with type="checkbox"
:submit $(":submit") All input elements with type="submit"
:reset $(":reset") All input elements with type="reset"
:button $(":button") All input elements with type="button"
:image $(":image") All input elements with type="image"
:file $(":file") All input elements with type="file"
:enabled $(":enabled") All enabled input elements
:disabled $(":disabled") All disabled input elements
:selected $(":selected") All selected input elements
:checked $(":checked") All checked input elements
     
:header $(":header") All header elements <h1>, <h2> ...
:animated $(":animated") All animated elements
:focus $(":focus") The element that currently has focus
:contains(text) $(":contains('Hello')") Tất cả các phần tử có nội dụng chưa từ "Hello"
:has(selector) $("div:has(p)") All <div> elements that have a <p> element
:empty $(":empty") All elements that are empty
:parent $("td:parent") Những <td> có phần tử con (chứa nội dung). Phần tử con có thể là văn bản, HTML, hoặc bất kỳ phần tử nào bên trong
:hidden $("p:hidden") All hidden <p> elements
:visible $("table:visible") All visible tables
:root $(":root") Chính là thẻ <html>
:lang(language) $("p:lang(de)") All <p> elements with a lang attribute value starting with "de"
     
:first-child $("p:first-child") Những thẻ <p> mà nó là con đầu tiên (first child) của phần tử nào đó
:first-of-type $("p:first-of-type") Những phần mà: Là <p> (vì cùng loại) và là con đầu tiên của một phần từ khác (tương đương :nth-of-type(1))
:last-child $("p:last-child") Những thẻ <p> mà nó là con cuối cùng (last child) của phần tử nào đó
:last-of-type $("p:last-of-type") Những phần mà: Là <p> (vì cùng loại) và là con cuối cùng của một phần từ khác
:nth-child(n) $("p:nth-child(2)") All <p> elements that are the 2nd child of their parent
:nth-last-child(n) $("p:nth-last-child(2)") All <p> elements that are the 2nd child of their parent, counting from the last child
:nth-of-type(n) $("p:nth-of-type(2)") All <p> elements that are the 2nd <p> element of their parent
:nth-last-of-type(n) $("p:nth-last-of-type(2)") All <p> elements that are the 2nd <p> element of their parent, counting from the last child
:only-child $("p:only-child") All <p> elements that are the only child of their parent
:only-of-type $("p:only-of-type") All <p> elements that are the only child, of its type, of their parent
